VARIÁVEIS

O que é?

As variáveis são como recipientes que armazenam dados. Esses dados podem ser de diversos tipos, como números, textos (strings), valores booleanos (true ou false), objetos e muitos outros. Ao declarar uma variável, você está reservando um espaço na memória do computador para guardar um valor.

Sintaxe

JavaScript é case-sensitive e usa o conjunto de caracteres Unicode. Por exemplo, a palavra Früh (que significa "cedo" em Alemão) pode ser usada como nome de variável. Mas a variável früh não é a mesma que Früh porque JavaScript é case sensitive.

Objetos

O que são objetos?

Em JavaScript, a maioria das coisas são objetos. Um objeto é uma coleção de dados e/ou funcionalidades relacionadas (que geralmente consistem em diversas variáveis e funções — que são chamadas de propriedades e métodos quando estão dentro de objetos). Vamos trabalhar com um exemplo para entender como eles são.

Por que usar objetos?

  • Organização:Agrupar dados relacionados em um único lugar torna o código mais organizado e fácil de entender.
  • Reutilização:Você pode criar vários objetos com a mesma estrutura, economizando tempo e evitando repetições.
  • Modelagem:Objetos permitem modelar entidades do mundo real, como pessoas, produtos, etc.

Como criar um objeto em JS?

JavaScript

                 
             // Criando um objeto literal
                  const pessoa = {
                      nome: "João", 
                      idade: 30,
                      cidade: "Brasília",
                      // Um método (função dentro do objeto)
                      saudacao: function() {
                        console.log("Olá, meu nome é " + this.nome);
                      }
                    };

Conceitos importantes

  • Propriedades:São os atributos do objeto, como nome, idade, marca, etc.
  • Métodos:São as ações que o objeto pode realizar, como saudação, ligar, desconto, etc.
  • This:Dentro de um método, this se refere ao próprio objeto
  • Construtor:Uma função especial usada para criar objetos de um determinado tipo.
  • Prototipos: Um mecanismo de herança em JavaScript que permite criar objetos que herdam propriedades e métodos de outros objetos.

Em resumo

Objetos são uma parte fundamental do JavaScript e são usados para modelar dados e comportamentos de forma organizada e flexível. Ao entender como criar e usar objetos, você estará habilitado a construir aplicações JavaScript mais complexas e poderosas

Estrutura de decisão

O que são as estruturas de decisão?

As estruturas de decisão permitem que o seu código tome caminhos diferentes com base em determinadas condições. Elas são essenciais para criar programas mais dinâmicos e inteligentes, que podem se adaptar a diferentes situações

Principais estruturas de decisão no JS

  • if...else: A mais básica, permite executar um bloco de código se uma condição for verdadeira, e outro bloco (opcional) se for falsa.
  • switch:Ideal para quando você precisa verificar o valor de uma expressão contra múltiplas opções.

Exemplo 1: if...else

JavaScript

 
               
                  let idade = 18,
                      if( idade >= 18)  {
                        console.log("Você é maior de idade." )
                      } else{ 
                       console.log("Você é menor de idade." )
                      };
  • A variável: idade armazena um número.
  • A condição: idade >= 18 é verificada.
  • Se a condição: for verdadeira (ou seja, a pessoa tem 18 anos ou mais), a mensagem "Você é maior de idade." é exibida no console.
  • Caso contrário: a mensagem "Você é menor de idade." é exibida.

Exemplo 2: switch

JavaScript

 
               
             
                  let diaDaSemana = "sábado",
                      switch( diaDaSemana)  {
                        case "sábado":
                        case "domingo":
                           console.log("Final de semana!" );
                        break   
                        case "segunda":
                        case "terça":
                        case "quarta":
                        case "quinta":
                        case "sexta":
                          console.log("Dia de semana!" );
                            break 
                        default:
                             console.log("Dia inválido." );

                      }
  • A variável: diaDaSemana armazena uma string.
  • O valor de: diaDaSemanaé comparado com cada case
  • Se o valor corresponder a um: case, o código dentro daquele bloco é executado.
  • O break interrompe a execução do switch após encontrar uma correspondência.
  • O default: é executado se nenhum dos case corresponder.

Laços de repetição

O que são laços de repetição?

Laços de repetição são estruturas que permitem executar um bloco de código repetidamente até que uma determinada condição seja atendida. Eles são essenciais para automatizar tarefas repetitivas e iterar sobre conjuntos de dados.

Tipos de laços de repetição:

  • for: Ideal quando você sabe o número exato de vezes que deseja repetir o bloco de código.
  • while:Executa o bloco de código enquanto uma condição for verdadeira.
  • do...while:Executa o bloco de código enquanto uma condição for verdadeira

Exemplo 1: Contando de 1 a 10 com o laço for

JavaScript

 
               
                  
      for( for i = 1; i <= 10; i++)  {
                        console.log(i )
                      } ;
  • i = 1: i com o valor 1.
  • i <= 10: É a condição para continuar o loop. Enquantoi for menor ou igual a 10, o loop continua.
  • i++: Incrementa o valor de iem 1 a cada iteração.

Calculando a soma dos números de 1 a 100 com o laço while

                 let count = 1
      while( numero <=  100)  {

                      numero++;

  • A variávelsoma armazena o valor 0
  • A variávelnumero armazena o valor 1.Essa variável será usada para controlar o loop e somar os números.
  • Swhile (numero <= 100): inicia um laço que se repete enquanto a condição for verdadeira.
  • soma += numero: A cada iteração do loop, o valor de numero é adicionado à variável soma O operador += é uma forma abreviada de soma = soma + numero;.

Loops: for in, for of e forEach

O que são esse loops?

Em JavaScript, temos diversas formas de iterar sobre arrays e objetos. Três das mais comuns são o for in, for of e forEach. Cada um possui suas particularidades e é mais adequado para determinadas situações.

for in

  • Uso:Iterar sobre as propriedades enumeráveis de um objeto.
  • Exemplo 
                 
                  const pessoa  = {
                      nome: "Marcio Araya", 
                      idade: 28,
                      cidade: "Brasília",
                  }
                      for(let propriedades in pessoa)  {
                        console.log(propriedade + ": " + pessoa[propriedade])
                        } ;

saida

 
               nome: Marcio Araya
               idade: 28
               cidade: Brasília

for of

  • Uso:Iterar sobre valores de arrays e objetos iteráveis (como strings, Map, Set).
  • Exemplo 
                 
                  const frutas  = ["maçã", "banana", "laranja"]
                      for(let fruta of frutas)  {
                        console.log(fruta)
                        } ;

saida

 
               maçã
               banana
               laranja

forEach

  • Uso:Executar uma função para cada elemento de um array.
  • Exemplo 
                 
                  const numeros  = [1, 2, 3, 4 ,5  ]
                      for(let fruta of frutas)  {
                        console.log(fruta)
                        } ;

saida

 
               maçã
               banana

Métodos de array no JS Map,filter e reduce

O que são esses métodos?

São funções de alta ordem que permitem iterar sobre os elementos de um array e realizar operações específicas em cada elemento.

Qual a função de cada um?

  • map(): Cria um novo array com os resultados da aplicação de uma função a cada elemento do array original. É como aplicar uma transformação a cada elemento
  • filter():Cria um novo array com todos os elementos que satisfazem uma determinada condição. É como filtrar os elementos de um array.
  • reduce():Reduz um array a um único valor, aplicando uma função acumuladora a cada elemento e um valor inicial. É útil para calcular totais, médias, etc.

Usando o map

JavaScript

 
                        
      const  numeros = [1, 2, 3 ,4, 5]
              //Dobrar cada número
              const  numerosDobrados = numeros.map(numero >= numer *2);
               console.log(numerosDobrados); //[2, 4, 6, 8, 10]
  • Criamos um array: com 5 elementos
  • O map(): percorre cada elemento do array numeros
  • A função anônima: numero => numero * 2 é aplicada a cada elemento. Essa função recebe o valor atual como numero e retorna o dobro desse valor.
  • O resultado: de cada aplicação da função é adicionado a um novo array, que é armazenado em numerosDobrados.

Filtrando números

JavaScript

 
                      
      const  numeros = [1, 2, 3 ,4, 5]
              //Filtrar números pares
              const  numerosPares = numeros.filter(numero >= numer % 2 === 0);
               console.log(numerosPares); //[2, 4,]
  • Criamos um array: novamente temos o mesmo array numeros
  • O método filter(): percorre cada elemento do array numeros
  • A função anônima: numero => numero % 2 === 0 é aplicada a cada elemento. Essa função retorna true se o número for par (resto da divisão por 2 é 0) e false caso contrário.
  • O filter(): cria um novo array com apenas os elementos para os quais a função retornou true

Somando todos os números

JavaScript

 
                      
      const  numeros = [1, 2, 3 ,4, 5]
              //Filtrar números pares
              const  soma  = numeros.reduce((total, numero) => total + numero, 0);
               console.log(soma); //15
  • Criamos um array: Novamente, temos o mesmo array numeros
  • O método reduce(): itera sobre o array, aplicando uma função a cada elemento e um acumulador que colocamos como 0
  • A função anônima: (total, numero) => total + numero recebe dois parâmetros: total (o valor acumulado) e numero (o elemento atual).
  • A função retorna a soma do total e do numero.
  • O valor inicial do acumulador é 0 (o terceiro argumento de reduce()).
