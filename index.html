VARIÁVEIS

O que é?

As variáveis são como recipientes que armazenam dados. Esses dados podem ser de diversos tipos, como números, textos (strings), valores booleanos (true ou false), objetos e muitos outros. Ao declarar uma variável, você está reservando um espaço na memória do computador para guardar um valor.

Sintaxe

JavaScript é case-sensitive e usa o conjunto de caracteres Unicode. Por exemplo, a palavra Früh (que significa "cedo" em Alemão) pode ser usada como nome de variável. Mas a variável früh não é a mesma que Früh porque JavaScript é case sensitive.

Objetos

O que são objetos?

Em JavaScript, a maioria das coisas são objetos. Um objeto é uma coleção de dados e/ou funcionalidades relacionadas (que geralmente consistem em diversas variáveis e funções — que são chamadas de propriedades e métodos quando estão dentro de objetos). Vamos trabalhar com um exemplo para entender como eles são.

Por que usar objetos?

  • Organização:Agrupar dados relacionados em um único lugar torna o código mais organizado e fácil de entender.
  • Reutilização:Você pode criar vários objetos com a mesma estrutura, economizando tempo e evitando repetições.
  • Modelagem:Objetos permitem modelar entidades do mundo real, como pessoas, produtos, etc.

Como criar um objeto em JS?

JavaScript

        

                 
             // Criando um objeto literal
                  const pessoa = {
                      nome: "João", 
                      idade: 30,
                      cidade: "Brasília",
                      // Um método (função dentro do objeto)
                      saudacao: function() {
                        console.log("Olá, meu nome é " + this.nome);
                      }
                    };

Conceitos importantes

  • Propriedades:São os atributos do objeto, como nome, idade, marca, etc.
  • Métodos:São as ações que o objeto pode realizar, como saudação, ligar, desconto, etc.
  • This:Dentro de um método, this se refere ao próprio objeto
  • Construtor:Uma função especial usada para criar objetos de um determinado tipo.
  • Prototipos: Um mecanismo de herança em JavaScript que permite criar objetos que herdam propriedades e métodos de outros objetos.

Em resumo

Objetos são uma parte fundamental do JavaScript e são usados para modelar dados e comportamentos de forma organizada e flexível. Ao entender como criar e usar objetos, você estará habilitado a construir aplicações JavaScript mais complexas e poderosas

Estrutura de decisão

O que são as estruturas de decisão?

As estruturas de decisão permitem que o seu código tome caminhos diferentes com base em determinadas condições. Elas são essenciais para criar programas mais dinâmicos e inteligentes, que podem se adaptar a diferentes situações

Principais estruturas de decisão no JS

  • if...else: A mais básica, permite executar um bloco de código se uma condição for verdadeira, e outro bloco (opcional) se for falsa.
  • switch:Ideal para quando você precisa verificar o valor de uma expressão contra múltiplas opções.

Exemplo 1: if...else

JavaScript

 
               
                  let idade = 18,
                      if( idade >= 18)  {
                        console.log("Você é maior de idade." )
                      } else{ 
                       console.log("Você é menor de idade." )
                      };
  • A variável idade armazena um número.
  • A condição idade >= 18 é verificada.
  • Se a condição for verdadeira (ou seja, a pessoa tem 18 anos ou mais), a mensagem "Você é maior de idade." é exibida no console.
  • Caso contrário, a mensagem "Você é menor de idade." é exibida.

Exemplo 2: switch

JavaScript

 
               
             
                  let diaDaSemana = "sábado",
                      switch( diaDaSemana)  {
                        case "sábado":
                        case "domingo":
                           console.log("Final de semana!" );
                        break   
                        case "segunda":
                        case "terça":
                        case "quarta":
                        case "quinta":
                        case "sexta":
                          console.log("Dia de semana!" );
                            break 
                        default:
                             console.log("Dia inválido." );

                      }
  • A variável diaDaSemana armazena uma string.
  • O valor de diaDaSemanaé comparado com cada case
  • Se o valor corresponder a um case,o código dentro daquele bloco é executado.
  • O break interrompe a execução do switch após encontrar uma correspondência.
  • O default é executado se nenhum dos case corresponder.
